## Blog Site
*Sugam's blog site*


 #### Start Project
 ```
    npm start

 ```

#### Run Project
 ```
    npm run dev

 ```

 #### Build Project
 ```
    npm run build

 ```

 ### Build Image

 ```
   docker build -t typescript-app .

 ```

### RUN Image as containers
```
   docker run -p 3000:3000 typescript-app

```
